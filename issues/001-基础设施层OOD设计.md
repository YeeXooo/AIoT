# 基础设施/适配层 OOD 设计

> 状态：待开始
> 前置：`docs/ood_domain.md` ✅ / `docs/ood_application.md` ✅

## 描述

为车载安全监测系统完成基础设施/适配层的架构级 OOD 设计文档。基础设施层负责数据持久化、仓储实现、领域事件总线、外部端口适配器、华为云服务集成、安全与隐私、边缘-云部署拓扑等横切技术关注点。

## 技术栈

- 后端：Java Spring Boot，部署华为云
- 数据库：金仓（兼容 PostgreSQL，JPA 方言 PostgreSQLDialect）
- 设备接入：华为云 IoTDA（MQTT）
- 推送：SMN
- 音视频：SparkRTC
- 前端：ArkTS（HarmonyOS）

## 产出要求

### 1. 数据持久化设计
- AR-01~AR-05 五个聚合根的数据库表结构（字段、类型、索引、主键策略）
- 值对象 `@Embeddable` 映射约定
- 乐观锁版本号字段约定

### 2. 仓储实现设计
- TripRepository / DriverRepository / VehicleRepository / SystemAccountRepository / RoadRageVoiceRecordRepository 的 JPA 实现要点
- 复杂查询的 JPQL/原生 SQL 策略（如按到期时间批量查询 RoadRageVoiceRecord）
- CQRS 读模型投影表的表结构与同步策略（告警历史查询、看板聚合查询）

### 3. 领域事件基础设施
- 事件总线选型（边缘侧单线程同步 EventBus + 云端 outbox + 消息队列异步投递）
- 事件持久化表结构、重试与死信策略
- 事件消费者注册机制

### 4. 外部端口适配器
VehicleStateBuffer、PhysiologicalDataBuffer、DrivingBehaviorTrackingPort、CameraOcclusionDetectionPort、OTADeliveryPort、NotificationPort、RescueReportPort 等的实现方案与技术选型。

### 5. 华为云服务集成
IoTDA 设备影子/Topic 路由、金仓数据库连接与分库分表（按 VehicleId 哈希分片）、SMN 消息模板、SparkRTC 房间/音频流管理。

### 6. 安全与隐私基础设施
数据脱敏边缘侧实现策略、语音存证边缘加密存储方案、二次身份验证集成方案。

### 7. 边缘-云部署拓扑
组件部署图、MQTT 信道拓扑、断网优雅降级策略。
